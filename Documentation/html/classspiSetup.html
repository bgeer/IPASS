Detailed Description

Class to make a spiBus.

This class make a spiBus with hwlib::spi_bus_bit_banged_sclk_mosi_miso. There are some basic Spi methods so you can easily communicate with the chip. This class uses hwlib (https://github.com/wovo/hwlib)
